# Press Contact Management System

This project is a corporate press contact and relationship management system developed using ASP.NET MVC.

## 🇹🇷 Türkçe Açıklama
Bu proje ASP.NET MVC ile geliştirilmiş kurumsal basın iletişim ve ilişki yönetim sistemidir.

## 🚀 Features
- User authentication and authorization
- Press contact management
- Event tracking
- Reporting system

## 🛠 Technologies
- ASP.NET MVC
- C#
- Entity Framework
- SQL Server

## 📂 Project Structure
- Presentation Layer
- Application Layer
- Business Layer
- Data Access Layer

## ⚙️ Setup
1. Clone the repository
2. Open solution in Visual Studio
3. Configure database connection
4. Run the project

## 📌 Notes
This project was developed for learning and portfolio purposes.
